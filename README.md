# 솔방이의 메모장
천우희를 사랑하는 펜심으로 만든 앱입니다. 
가볍게 메모를 할 때도 쓸 수 있으며, 힘들고 지칠 때 100장의 천우희의 귀엽고 이쁜 짤방을 보면서 힘을 낼 수 있는 앱입니다. 우리 모두 천우희씨를 보면서 힘든 인생을 힘내봅시다. 

## 구동방법
